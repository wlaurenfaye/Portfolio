Projects

I am currently a student at Liverpool John Moores University studying undergradute course of Multimedia Computing. This course involves multimedia modules which has so far covered the fundamentals in multimedia and problem solving in multimedia. I also have common computing modules which have so far included introduction to programming in java; computer systems; internet and web technologies and data modelling.

Computer Systems Project

For this project in my first semester I used Digital Works 95 to produce a circuit which would use an LED output to write my student ID number. It included a fully labelled diagram of the circuit which has been drawn using Digital Works 95, the Boolean equations for the circuit and n extended truth table. This was a very practical based module, I learned a lot about binary code, circuits and equations. My mark was 91/100.

Multimedia Project

In the last semester my “Fundamentals in Multimedia” module has done just that in laying the fundamental knowledge needed for my development as a multimedia development. Having very little to no experience in Adobe’s Photoshop and Illustrator software’s, the lectures and tutorials I have taken part in this semester have given me a range of skills and knowledge in this area that I am keen to develop further. Adobe Photoshop is a vital piece of software in any media-based career, which has been used by developers for 15 years with such a diverse set of abilities it can be used from editing photographs to creating company logos. For the module coursework I produced a portfolio with examples using the skills of photoshop and illustrator I had learned, including photo editing; icon creating; 3D object creation and some simple 2D animation.
